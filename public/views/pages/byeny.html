ByeNY

ByeNY was a projected built during my studies at General Assembly's Web Development Immersive program. This project utilizes data from flightstats.com developers API and displays the data using D3.js. The data taken are all out bound flights during a 24 hour period starting at 12 midnight the night before. Data must be from the previous day because current day is not available. The colors denote which continent flights are departing to. Please note, this graphs flight paths in a linear fashion to their destination. In reality howeer, flight paths rarely follow a linear path. Check it out here.

Fre-ebooks

Fre-ebooks is a side project I created for the students in my Front End Web Development class. It aggregrates all the data on free programming ebooks from Github and displays them in the library section of the site. In this project, I played with some new technologies such as Angular, Express and Nodejs. Originally the project started out as using the Github API however all the data was stored in a readme.md file. This would not parse well when using Github's API so I used a tool called Marked NPM which turns all markup language into formatted HTML. From there, it would display nicely on the page. Feel free to check it out here.
